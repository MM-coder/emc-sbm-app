About the Platform

I made this to factilitate communication between all of the members that participate in these monthy student-teacher meetings

The platform is coded in HTML + CSS with a Flask (Python) backend the frontend uses uikit for styling. The only JS library used was jquery (only on this page)

About me

Hi, I'm Mauro, I've got approximately 7 years experience in programming and I took up the project for the aforementioned reasons. I take on every project as a learning oportunity and this one, for school, was no different.
